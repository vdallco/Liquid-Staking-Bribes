What are LSD Bribes?
  • currency_exchange Stakehouse LSD Validators deposit ERC-20 tokens to incentivize stakers to deposit ETH directly to their validator via Fren Delegation
  • redeem Stakers who use Stakehouse Fren Delegation to deposit ETH directly to incentivized validators earn ERC-20 token rewards.
  • hub Stakehouse LSD Network owners deposit ERC-20 tokens to incentivize Node Runners to join their network
  • redeem Node operators who join incentivized LSD Networks earn ERC-20 token rewards.

Warning
  • Use the lsd-mainnet graph to confirm you have the correct validator BLS key or LSD Network ticker.
  • Beware of unverified reward tokens. Tokens with a green checkmark are verified to be legitimate. Always verify the reward token by checking the Etherscan link.
